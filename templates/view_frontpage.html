Youtube browser upload demo

Hi! This is the sample implementation of Youtube API for Python.

It uses Flask, Google Gdata client and Javascript/Flash uploader.

This project just a non-commercial mix of libraries and APIs.
All sources (with third-party libraries) at Github. Project works in local machine and GAE both (yaml-files included).

So, you can upload video files to Youtube with browser to special demo account.
